# Projelerim

Kocaeli Üniversitesi Bilgisayar Mühendisliği dönem projelerinden örnekler (arşiv).

## Big-O Analizi (`big-o/`)

Verilen kaynak kod dosyasını okuyup döngü kalıplarını tespit ederek algoritmanın Big-O karmaşıklığını (O(n), O(n²) vb.) tahmin eden C programı. Programın kendi çalışma süresini de `clock()` ile ölçer.

- **Dil:** C (Code::Blocks projesi)
- Girdi olarak `myfile.txt` içindeki kodu analiz eder; sözde kod örneği `yalancı kod.txt` dosyasındadır.

## Summary Text Extractor (`Summary Text Extractor Program/`)

Metin özetleme/çıkarma uygulamasının proje raporu (IEEE formatında) ve uygulamadan ekran görüntüleri. Yazılım Laboratuvarı II kapsamında grup projesi olarak geliştirilmiştir.
